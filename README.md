## SETUP
```bash
1. Clone github repository using git clone https://github.com/loghogjog/INF1004-P8-9.git
2. Install tailwind CSS standalone CLI from https://github.com/tailwindlabs/tailwindcss/releases/tag/v4.1.13 # download to your project directory and rename to tailwindcss
3. pip install -r requirements.txt
4. ./tailwindcss -i static/src/input.css -o static/dist/output.css # build your own output.css
5. python app.py # runs app.py
6. Use five server to run index.html
```
