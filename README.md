## SETUP
```bash
1. Clone github repository using git clone https://github.com/loghogjog/INF1004-P8-9.git
2. Use either pip or pip3 for 3.
3. Install Flask by typing in the terminal pip install flask
3.5. Install Jinja2 if not already installed pip install jinja2
4. Install tailwind CSS standalone CLI from https://github.com/tailwindlabs/tailwindcss/releases/tag/v4.1.13 # download to your project directory and rename to tailwindcss
5. pip install -r requirements.txt
6. ./tailwindcss -i static/src/input.css -o static/dist/output.css # build your own output.css as tailwind compiles differently for different OS
7. python app.py # runs app.py
8. Use Ctrl + S to reload the debugger after editing your code
```
