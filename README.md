## SETUP
```bash
1. Install tailwind CSS standalone CLI from https://github.com/tailwindlabs/tailwindcss/releases/tag/v4.1.13 (if you download to project directory, make sure to rename to tailwindcss)
2. pip install -r requirements.txt
3. cd to your project directory
4. <PATH_TO_EXECUTABLE> -i src/input.css -o src/output.css --watch
5. python app.py
```
