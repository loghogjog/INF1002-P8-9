## SETUP
```bash
1. Install tailwind CSS standalone CLI from https://github.com/tailwindlabs/tailwindcss/releases/tag/v4.1.13 # download to your project directory and rename to tailwindcss
2. pip install -r requirements.txt
3. ./tailwindcss -i static/src/input.css -o static/dist/output.css # build your own output.css
4. python app.py # runs app.py
5. Use five server to run index.html
```
